按学习方式分类

机器学习算法按照学习方式的不同，可以分为以下四大类型：

Supervised Learning
监督学习
输入数据有标签，模型通过反馈不断修正，目的是预测结果。
Unsupervised Learning
无监督学习
输入数据无标签，模型自主发现数据的内部结构和隐藏模式。
Semi-supervised Learning
半监督学习
部分有标签、部分无标签，降低数据标记难度，兼顾效率与效果。
Reinforcement Learning
强化学习
智能体与环境交互，通过试错获取最佳策略，追求长期利益最大化。
Key
四种学习方式的核心区别在于：训练数据是否有标签，以及模型的学习目标是什么。

监督学习 — 回归与分类

监督学习的核心特征：输入数据有标签，输出有反馈，目的是预测结果。模型学习特征到标签的映射函数 f: X → Y。

监督学习包含两大子任务：

回归 Regression
输出是连续值
预测结果为数值型
例：房价预测 → 320.5万
模型输出可以是任意实数
分类 Classification
输出是有限个离散值
预测结果为类别标签
例：猫狗分类 → 猫 / 狗
模型输出是预定义类别之一
典型案例
分类：猫狗分类、垃圾邮件识别、疾病诊断
回归：房价预测、气温预测、销量预估
关键区别：回归的输出空间是连续的（无穷多个值），分类的输出空间是离散的（有限个类别）。

无监督学习 — 聚类

无监督学习的训练数据没有标签，但必须有特征。模型的任务不是预测，而是发现数据的内部结构及样本间的相互关系。

聚类是无监督学习最典型的方法：根据样本之间的特征相似性，将数据自动分成若干组。

1
数据无标签但有特征 每个样本都有可度量的特征值，但没有人告诉机器"这是什么类别"。
2
按特征相似性聚类 模型计算样本间的距离或相似度，将相近的样本归为一组。
3
发现内部结构 聚类揭示数据的潜在分布，发现事物的内部结构及相互关系。
4
结果与超参数有关 没有标签就没有标准固定归类，聚类结果受超参数（如簇数 K）影响。
古语类比
"物以类聚，人以群分" — 这正是无监督聚类的核心思想：相似的事物自然归为一类。

半监督学习与强化学习

半监督学习 Semi-supervised Learning

训练数据中部分有标签、部分无标签。利用少量标注数据和大量未标注数据共同训练模型。

核心优势：大幅降低数据标记难度和成本。在标注数据稀缺的场景下尤为重要 — 真实世界中，获取标签往往代价高昂。

强化学习 Reinforcement Learning

智能体 (Agent) 与环境 (Environment) 持续交互，通过试错获取最佳策略，目标是长期利益最大化。

四大要素：

Agent
智能体
Environment
环境
Action
行动
Reward
奖励
案例
半监督：医学影像分析（少量专家标注 + 大量未标注影像）
强化学习：学下棋、机器人控制、自动驾驶决策

四种学习方式对比 & 数学表示

类型 输入 (In) 输出 (Out) 目的 案例
监督学习 有标签 有反馈 预测结果 猫狗分类、房价预测
无监督学习 无标签 无反馈 发现结构 聚类分组
半监督学习 部分有标签 有反馈 降低标记难度 医学影像分析
强化学习 决策流程 一系列行动 长期利益最大化 学下棋

数学表示：

监督学习
输入 X 含特征值
输出 Y 含目标值
学习 X → Y 的映射函数
无监督学习
输入 X 含特征值
无目标值 Y
学习 X 的内部结构
本质区别：监督学习有 Y 作为"老师"指导学习方向；无监督学习没有 Y，模型只能自主探索数据分布。

章节小结

  • 四种学习方式：机器学习按学习方式分为监督学习、无监督学习、半监督学习和强化学习。
  • 监督学习：数据有标签，包含回归（输出连续值）和分类（输出离散值）两大子任务。
  • 无监督学习：数据无标签但有特征，通过聚类发现数据的内部结构和相互关系。聚类结果与超参数有关，没有标准固定归类。
  • 半监督学习：部分有标签 + 部分无标签，核心优势是降低数据标记成本。
  • 强化学习：Agent 与 Environment 交互，通过试错获取最佳策略。四要素：智能体、环境、行动、奖励。
  • 数学表示：监督学习的输入 X 含特征值、输出 Y 含目标值，学习 X→Y 映射；无监督学习输入 X 含特征值，无 Y，学习 X 的内部结构。
一句话概括
机器学习算法分类的核心在于：数据是否有标签，以及学习目标是什么。
