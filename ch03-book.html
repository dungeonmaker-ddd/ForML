按学习方式分类

机器学习算法按照学习方式的不同，可以分为以下四大类型。核心区别在于训练数据是否有标签，以及学习目标的差异。

Supervised Learning
监督学习
输入：有标签
输出：有反馈
目的：预测结果
案例：猫狗分类、房价预测
Unsupervised Learning
无监督学习
输入：无标签（但有特征）
输出：无反馈
目的：发现潜在结构
案例：物以类聚、人以群分
Semi-supervised Learning
半监督学习
输入：部分有标签、部分无标签
输出：有反馈
目的：降低数据标记难度
Reinforcement Learning
强化学习
输入：决策流程及激励系统
输出：一系列行动
目的：长期利益最大化
案例：学下棋
Glossary
监督学习Supervised Learning
无监督学习Unsupervised Learning
半监督学习Semi-supervised Learning
强化学习Reinforcement Learning

监督学习 — 回归与分类

监督学习的数据同时包含特征 X 和标签 Y，有明确的正确答案。训练时预测值与真实标签对比，计算 loss 并调整参数。监督学习包含两大子任务：

回归 Regression
函数输出是连续值
如：房价预测
预测的目标是一个具体数值
输出空间是连续的数值范围
分类 Classification
函数输出是有限个离散值
如：猫狗分类
预测的目标是某个类别
输出空间是有限的离散类别
Key
回归与分类的核心区别在于输出值的性质 — 连续值 vs 离散值。两者都属于监督学习，都需要有标签的数据进行训练。
Glossary
回归 Regression输出连续值
分类 Classification输出有限个离散值

无监督学习 — 聚类

无监督学习的数据没有标签 Y，但必须有特征。没有标准正确答案，训练过程无反馈，无法直接评判对错。

聚类 (Clustering) 是无监督学习的典型任务 — 按特征相似性对样本进行自动分组。聚类没有标准固定归类，结果与超参数的设定密切相关。

核心目标
发现事物的内部结构及样本间的相互关系，揭示数据中隐藏的模式和规律。
"物以类聚，人以群分" — 即使没有人告诉机器"这是什么"，机器也能通过特征的相似性将样本自动分组，发现数据中潜在的结构。
Glossary
聚类 Clustering按特征相似性分组
超参数 Hyperparameter人为设定的控制参数

半监督学习与强化学习

半监督学习是监督学习与无监督学习的折中方案 — 数据中部分有标签、部分无标签，核心目的是降低数据标记的难度，减少标注成本。

半监督学习
同时包含有标签和无标签数据，介于监督与无监督之间。用少量标注数据引导大量未标注数据的学习。

强化学习的输入是决策流程及激励系统，输出是一系列行动，目标是长期利益最大化。它由四个核心要素构成：

Agent
智能体
做出决策的主体
Environment
环境
智能体所处的世界
Action
行动
智能体的每次操作
Reward
奖励
环境对行动的反馈

智能体通过不断试错获取最佳策略，如学下棋。

Glossary
智能体 Agent做出决策的主体
奖励 Reward环境对行动的反馈信号

类型 输入 In 输出 Out 目的 案例

章节小结

  • 四种学习方式：机器学习按学习方式分为监督学习、无监督学习、半监督学习和强化学习，核心区别在于数据中是否包含标签。
  • 监督学习两大子任务：回归输出连续值（如房价预测），分类输出离散值（如猫狗分类）。
  • 无监督学习：数据无标签但有特征，通过聚类按特征相似性分组，发现数据的内部结构和相互关系。结果与超参数有关，没有标准固定归类。
  • 半监督 & 强化学习：半监督用部分标签降低标注成本；强化学习以 Agent / Environment / Action / Reward 四要素为框架，通过试错获取最佳策略。
  • 数学形式：监督学习学习映射函数 f : X → Y；无监督学习学习 X 的内部结构与分布。
