  JSON 格式

  C:\Users\Administrator\.claude\projects\C--Users-Administrator-Desktop-ai-ML\
  2. 里面有 JSONL 格式的会话记录文件

  手动方式

  3. 启动时加 --output-format json 可以把整个会话以结构化格式输出
  4. 用 claude conversation list 和 claude conversation resume
  可以查看和恢复历史会话

  最实用的做法

  直接在对话里输入 /export，选择保存路径，就能拿到完整的对话文本。
