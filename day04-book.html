正规方程 vs 梯度下降 — 什么时候换

小弟
Claude的小弟
Day 03 学了正规方程一步出结果，那梯度下降存在的意义是什么？正规方程不是更好吗？
C
Claude
正规方程的瓶颈在矩阵求逆 (XTX)−1，复杂度是 O(n³)。n 翻倍，计算量变 8 倍。梯度下降对 n 只是线性 O(n)，翻倍只变 2 倍。
复杂度n=10000 时
正规方程O(n³ + m·n²)1012 + 1014
梯度下降O(m·n·k)，k=1000109 × 103 = 1012
小弟
Claude的小弟
那特征数没那么多的时候也可以用梯度下降吧？
C
Claude
可以，结果几乎一样（MSE 是凸函数，全局最优只有一个）。但多了调学习率、迭代次数的麻烦。相当于家离超市 200 米，走路 3 分钟到，你非要开车 — 能到，但还得找车位。
C
Claude
学梯度下降不是为了线性回归本身。逻辑回归没有正规方程，神经网络没有正规方程。后面所有模型的优化方式都是梯度下降，线性回归只是最简单的练习场。
Key — 判断流程
特征数 n > 10000？→ 梯度下降（SGDRegressor）
需要在线学习？→ 梯度下降（partial_fit）
都不是？→ 正规方程（LinearRegression）就够了
Glossary
Batch GD 全梯度下降，每步用全部样本，精确但慢
SGD 随机梯度下降，每步用 1 个样本，快但抖
Mini-batch 小批量，每步 64 个样本，工业默认选择

计算过程 — 正向传播与反向传播

小弟
Claude的小弟
梯度下降的计算过程是什么样的？应该没有脱离人类计算的范畴吧？
C
Claude
完全没有，每一步都是加减乘除。先正向算 loss，再反向算梯度。
1
正向 — 算预测ŷ = w·x + b = 1×2 + 3 = 5
2
正向 — 算误差e = ŷ − y = 5 − 7 = −2
3
正向 — 算损失L = e² = (−2)² = 4
4
反向 — 链式法则∂L/∂e = 2e = −4   ∂e/∂z = 1   ∂z/∂w = x = 2
5
乘起来dw = −4 × 1 × 2 = −8   db = −4 × 1 × 1 = −4
6
更新参数w = 1 − 0.1×(−8) = 1.8   b = 3 − 0.1×(−4) = 3.4
小弟
Claude的小弟
链式法则是什么？L 不直接含 w，怎么对 w 求导？
C
Claude
L 不认识 w，但 L 认识 e，e 认识 z，z 认识 w。一层层问：L 对 e 怎么变？e 对 z 怎么变？z 对 w 怎么变？每层的“放大倍数”乘起来就是答案。
∂L/∂w = ∂L/∂e × ∂e/∂z × ∂z/∂w = 2e × 1 × x
每层都是最基础的求导：e²→2e   z−y→1   wx→x
链式法则只是把它们串起来的方式
Key — 多样本 & 多维推广
多样本：每个样本算一组 dw、db，取平均
多维：n 个特征 → n 个 dw，每个 dwj = 误差 × 对应的 xj
矩阵写法：dw = 1/m × XT @ e — 不管 n 多大，代码不变
Glossary
Forward 正向传播，从参数算到 loss
Backward 反向传播，从 loss 算回梯度 dw、db
Chain Rule 链式法则，嵌套函数逐层求导再相乘

学习率与标准化 — 两个必调的旋钮

小弟
Claude的小弟
学习率就是对算出来的 dw 和 db 乘这个值吗？
C
Claude
对。w = w − α × dw，α 就是那个乘数。同样 dw=−8：α=0.01 变化 0.08（小碎步），α=0.1 变化 0.8（正常步），α=10 变化 80（飞出去了）。
α 合适
loss 稳步下降后走平
w: 0 → 1.2 → 2.8 → 4.7 → 5.0 ✓
α 太大
loss 震荡或爆炸
w: 0 → 8 → −3 → 12 → NaN ✗
小弟
Claude的小弟
梯度下降的标准化和正规方程的标准化不一样吗？
C
Claude
操作完全一样，都是 StandardScaler。区别是：正规方程不需要标准化（直接解方程不受量级影响），梯度下降必须标准化。因为梯度 = 误差 × 特征值，特征值大的方向梯度就大、步子就大，各方向不均匀就会震荡。
# ═══ 正规方程（不需要标准化）═══ model1 = LinearRegression() model1.fit(X_train, y_train) # ═══ 梯度下降（必须标准化）═══ scaler = StandardScaler() X_train_s = scaler.fit_transform(X_train) X_test_s = scaler.transform(X_test) model2 = SGDRegressor(max_iter=1000, eta0=0.01) model2.fit(X_train_s, y_train) # 结果几乎相同，R² 差距 < 0.01
Key — 标准化 ≠ 治异常值
标准化管特征间的尺度差异（面积 vs 收入）
异常值管样本间的偏离程度（房价 30 万 vs 打错的 300 万）
标准化后异常值还是异常值，只是数字刻度变了
Glossary
α (eta) 学习率，控制参数每步更新幅度
loss curve 损失曲线，判断学习率是否合适的核心工具

评估指标 — R² + RMSE 双视角

小弟
Claude的小弟
评估的方式有几种？都是对 w 和 b 进行评估的吗？
C
Claude
不是评估 w 和 b 本身，是评估用这组 w 和 b 算出来的预测准不准。你不关心 w 是 2.3 还是 2.7，只关心预测偏差有多大。
指标公式看什么用途
MSE1/m × Σ(ŷ−y)²放大大误差，单位 y²损失函数
RMSE√MSE偏差约几万，好解释汇报结果
MAE1/m × Σ|ŷ−y|不放大，一视同仁有异常值时
R²1 − MSE/Var比猜平均值好多少整体判断
小弟
Claude的小弟
R 平方是什么？跟其他指标有什么区别？
C
Claude
MSE/RMSE/MAE 告诉你“偏了多少”，但你不知道好不好 — RMSE=3 万在 500 万房价里很准，在 10 万房价里太离谱。R² 不看具体偏差，看“你比无脑猜平均值好了百分之几”，0.95 就是消除了 95% 的误差。
R² = 1 − Σ(ŷi − yi)² / Σ(ȳ − yi)²
你的模型误差 / 猜平均值的误差   R²=1 完美   R²=0 跟猜一样差   R²<0 比猜还差
Key — 两层评估
训练过程中：看 loss 曲线（MSE 随迭代次数变化）→ 学习率对不对
训练完之后：看 R² + RMSE → 模型好不好
先保证 loss 曲线正常收敛，再看评估指标才有意义
Glossary
R² 决定系数，0~1 刻度，不同数据集间可比
Cross-Val 交叉验证，K 折轮流测试，小数据集更可信

拟合诊断与正则化 — L1 & L2

小弟
Claude的小弟
怎么判断模型是欠拟合还是过拟合？
C
Claude
比两个数字：训练集 R² 和测试集 R²。
训练集 R²测试集 R²诊断操作
0.500.48欠拟合加多项式 / 加特征 / 换模型
0.900.88正好不用动
0.990.65过拟合加正则化 / 减特征 / 加数据
小弟
Claude的小弟
正则化是如何影响学习的？
C
Claude
在 loss 里加一个惩罚项：L = MSE + λ × (w 的大小)。误差推着 w 往上走，正则化拉着 w 往下 — 就像地心引力，w 越大引力越强，最终在平衡点稳定。
L2 Ridge — 弹簧
惩罚 Σwj²
w 大 → 拉力大
w 小 → 拉力也小
全部变小，不会归零
特征都有用时选
L1 Lasso — 刀
惩罚 Σ|wj|
不管 w 多大多小
每次砍固定的量
小的 w 直接归零 = 自动选特征
怀疑有垃圾特征时选
小弟
Claude的小弟
正则化只在训练过程中起作用吗？
C
Claude
对，只介入两处：loss 多了 λ×Σw² → dw 自动多了 λ×w。正向传播不变，更新公式不变，训练完之后不参与。就像引力只在球上升时拉它，球落地后任务就完成了。
from sklearn.linear_model import Ridge, Lasso Ridge(alpha=1.0) # L2: 全部压小，不归零 Lasso(alpha=0.1) # L1: 没用的 w 直接归零 # alpha 就是 λ，控制惩罚力度 # λ=0 无惩罚(可能过拟合) | 适中刚好 | 太大欠拟合
Glossary
Ridge L2 正则化，弹簧，全部保留全部压小
Lasso L1 正则化，刀，自动特征选择
ElasticNet L1+L2 混合，不确定时用

Day 04 · Summary

章节小结

C
Claude
来，今天学了很多，梳理一下核心知识：
  1. 正规方程 vs 梯度下降 — 不是递进关系，n > 10000 或需要在线学习时换梯度下降。
  2. 计算过程 — 正向传播算 loss → 反向传播算 dw（链式法则）→ 更新参数。每步都是加减乘除。
  3. 学习率 α — 看 loss 曲线调。稳步下降=合适，震荡=太大，龟速=太小。
  4. 标准化对梯度下降必须 — 让各方向步长均匀，直奔谷底不震荡。
  5. 评估用 R² + RMSE — R² 定性判断好不好，RMSE 定量感受偏差多大。
  6. 拟合诊断 — 比训练集和测试集的 R²。都低=欠拟合，训练高测试低=过拟合。
  7. 正则化 = 给 w 加引力 — L2 弹簧压所有人，L1 刀砍弱者。
Error Log — 易错清单
• loss 曲线管“训练过程对不对”，R² 管“模型选择对不对”— 两个层面不能混
• 标准化治不了异常值 — 标准化管特征间尺度，异常值管样本间偏离
• 一次方程调到死也达不到三次的效果 — 这是模型能力上限不是训练问题
• R² 不是“误差占平均值的比”— 是你的模型误差 vs 猜平均值误差的比
w = w − α · dw   dw = 1/m · XT(ŷ − y)
R² = 1 − Σ(ŷ−y)² / Σ(ȳ−y)²  |  Ridge: L + λΣw²  |  Lasso: L + λΣ|w|
