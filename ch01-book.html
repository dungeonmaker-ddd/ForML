人工智能 (AI) 是最大的范畴，涵盖一切让机器表现出智能行为的技术。

机器学习 (ML) 是 AI 的核心分支 —— 让机器自动从数据中学习规律，而非由人类编写规则。

深度学习 (DL) 是 ML 的子集，专指基于神经网络的学习方法族。

Key
AI 包含 ML，ML 包含 DL。三者是严格的嵌套包含关系。

机器学习的本质 — 从数据中自动学习

机器学习的核心思想：利用数学表达式，自动地从数据中获取规律，产生模型。这就是"机器自动学习"。

基于规则的系统
人工编写 if-else 规则
规则由专家定义
数据只用于查询
难以应对复杂场景
机器学习
从数据中自动提取规律
模型由数据驱动生成
数据用于学习和预测
能处理高维复杂问题
本质区别
基于规则 → 人告诉机器怎么做
机器学习 → 数据告诉机器规律是什么

ML 包含的典型算法：回归、分类、聚类、决策树、贝叶斯模型、支持向量机、自适应增强算法、最大期望算法等。

核心概念 — 特征、标签与样本

以房价预测为例 —— 根据房屋面积，预测房屋售价：

# 面积 (m²) 售价 (万元)
160120
280160
3100210
4120260
5150330

← 横向一行 就是一个样本 (sample)。整张表是一个数据集。

↑ 面积这一列 是特征 (feature) —— 模型的输入。

↑ 售价这一列 是标签 (label) —— 模型要预测的目标。

输入 → 模型 → 输出
面积（特征） → 学习到的模型 → 售价（标签）
特征是已知条件，标签是预测目标。

线性模型 — y = wx + b

将样本画入坐标系，可以看到数据点的分布规律。我们需要一条直线来描述这个规律：

y = wx + b
其中 w 和 b 是需要通过学习获得的模型参数
w
斜率 (weight)
控制直线的倾斜程度
w 越大，面积对售价影响越大
决定特征与标签的关联强度
b
截距 (bias)
控制直线的上下平移
表示面积为 0 时的基准售价
调整模型的整体偏移量
要点
w 和 b 不是人为设定的，而是机器从数据中自动学习得到的。这正是"机器学习"的含义。

从数据点到模型预测

线性模型的完整学习过程分四步：

1
绘制数据点 将每个样本画入坐标系（横轴：面积，纵轴：售价），得到 n 个数据点，观察分布趋势。
2
选择模型 观察到近似线性分布，选择线性模型 y = wx + b 来描述这个规律。
3
训练学习参数 让机器自动调整 w 和 b，使直线尽可能贴合所有数据点。模型"学会"了数据中的规律。
4
开始预测 训练完成后，输入一个新的面积值 x，模型输出预测售价 y。这就是模型的实际应用。

章节小结

  • 三层嵌套：AI 是最大范畴，ML 是 AI 的核心分支，DL 是 ML 中基于神经网络的方法族。
    AI ⊃ ML ⊃ DL
  • ML 的本质：自动从数据中获取规律、产生模型。与基于规则的系统有本质区别 —— 数据驱动 vs 规则驱动。
  • 核心术语：输入是特征 (feature)，输出是标签 (label)，一行数据是一个样本 (sample)。
  • 线性模型：y = wx + b。w 控制倾斜度（斜率），b 控制上下平移（截距）。两者均由机器自动学习。
  • 学习流程：数据点 → 选择模型 → 训练参数 → 预测新值。模型能够描述数据分布，训练好后即可用于预测。
一句话概括
机器学习就是让机器自动从数据中找到 w 和 b，用数学模型描述世界的规律。
