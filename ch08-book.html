KNN 核心思想

KNN 的核心思想极其直观：根据离你最近的 K 个邻居的类别，来判断你属于哪个类别。俗话说"物以类聚，近朱者赤"。

KNN 是一种惰性学习算法 —— 没有显式的训练过程，预测时才计算距离做判断。所有计算发生在预测阶段。

K 值的影响
K 太小：容易受噪声影响，模型过于敏感 → 过拟合
K 太大：分类界限变得模糊，模型过于粗糙 → 欠拟合
K 值的选择直接决定模型效果，是 KNN 最重要的超参数。
Terms
KNNK-Nearest Neighbors，K 近邻算法
Lazy Learning惰性学习，不主动训练，预测时才计算
Hyperparameter超参数，训练前人为设定的参数，如 K 值

分类与回归 — 处理流程与 API

KNN 既能处理分类问题，也能处理回归问题，区别仅在最后一步的汇总方式：

1
计算距离 计算待预测点与所有训练样本之间的距离。
2
排序取 K 个 按距离从小到大排序，取前 K 个最近邻。
3
汇总决策 分类：投票法（少数服从多数）→ 输出离散类别
回归：取均值 → 输出连续数值
KNeighborsClassifier
sklearn.neighbors
用途：分类任务
决策：K 个邻居投票
关键参数：
· n_neighbors = K 值
· algorithm = 距离算法
KNeighborsRegressor
sklearn.neighbors
用途：回归任务
决策：K 个邻居均值
关键参数：
· n_neighbors = K 值
· algorithm = 距离算法
回顾 Ch03
分类输出离散值（如猫/狗），回归输出连续值（如房价）—— 这是监督学习的两大子任务。KNN 两者都能做。
Terms
Classification分类，输出离散类别标签
Regression回归，输出连续数值
Voting投票法，KNN 分类中少数服从多数的决策方式

距离度量方法

KNN 的核心在于计算距离。不同的距离公式衡量样本间"相似度"的方式不同：

d = √∑(xi − yi)²
欧氏距离 (Euclidean) — 两点间直线距离，最常用
d = ∑|xi − yi|
曼哈顿距离 (Manhattan) — 各维度差的绝对值之和，如城市街区路径
d = (∑|xi − yi|p)1/p
闵可夫斯基距离 (Minkowski) — 通用推广：p=1 即曼哈顿，p=2 即欧氏
Key
距离计算受量纲影响极大。如果一个特征的量级是万、另一个是个位数，量级大的特征会主导距离结果。所以 KNN 必须做特征预处理。
Terms
Euclidean欧氏距离，两点间直线距离
Manhattan曼哈顿距离，城市街区式路径距离
Minkowski闵可夫斯基距离，欧氏与曼哈顿的推广

特征预处理 — 归一化与标准化

KNN 基于距离计算，量纲差异会严重影响结果。量纲大的特征主导距离计算，量纲小的特征被忽略，导致结果失真。

MinMaxScaler
归一化
缩放到 0 ∼ 1 范围
公式：x' = (x − min) / (max − min)
适用：已知数据范围、无明显异常值
StandardScaler
标准化
转换为均值 0、标准差 1
公式：x' = (x − mean) / std
适用：数据有异常值、大多数 ML 任务
选择策略
不确定时先试标准化，再对比归一化效果。标准化对异常值更鲁棒，是多数场景的安全默认选择。
鸢尾花识别案例
用 KNN 分类鸢尾花（Iris）数据集，对比预处理前后的分类效果。特征预处理能显著提升 KNN 分类准确率。
回顾 Ch05
特征预处理是特征工程五大子领域之一。归一化和标准化是解决量纲问题的两种核心方法，在 KNN 这种基于距离的算法中尤为关键。
Terms
MinMaxScaler归一化，将数据缩放到 [0, 1] 范围
StandardScaler标准化，转换为均值 0 标准差 1
Iris鸢尾花数据集，经典的分类入门数据集

章节小结

  • KNN 思想：根据最近的 K 个邻居判断类别，"物以类聚，近朱者赤"。惰性学习，无显式训练过程。
  • 分类与回归：分类用投票法，回归用均值法。sklearn 提供 KNeighborsClassifier 和 KNeighborsRegressor 两个 API。
  • 距离度量：欧氏距离最常用。曼哈顿距离和闵可夫斯基距离是替代和推广。
  • 特征预处理：KNN 基于距离，量纲差异严重影响结果。归一化 (MinMaxScaler) 和标准化 (StandardScaler) 是必须步骤。
  • 超参数选择：交叉验证可靠评估模型，网格搜索穷举找最优 K 值。两者配合是调参标准流程。
一句话概括
KNN 用距离找最近邻、用投票或均值做决策；效果取决于距离度量、特征预处理和K 值选择三个关键环节。
