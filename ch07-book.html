有监督学习和无监督学习的各自特点及区别

监督学习 Supervised
数据同时包含特征 X 和标签 Y，有明确的正确答案
学习映射函数 f : X → Y，目标是预测新样本的标签
训练有反馈：预测值与真实标签对比，计算 loss，调整参数
两大子任务：回归（输出连续值，如房价）& 分类（输出离散值，如猫狗）
无监督学习 Unsupervised
数据只有特征 X，没有标签 Y，没有标准正确答案
学习数据内部结构和分布规律，发现样本间相似性和隐藏模式
训练无反馈：无标签，无法直接评判对错
典型任务：聚类（按相似性自动分组），结果与超参数有关
对比维度 监督学习 无监督学习
数据形态 有标签、有反馈 无标签、无反馈
目标导向 预测已知类别的结果 发现未知的内部结构
评估难度 可用准确率、loss 直接评估 缺乏标准答案，评估困难
案例 标注好的猫狗图片 → 学会区分 未标注图片 → 按相似性自动分组
核心区别
根本区别在数据形态：监督学习有标签有反馈，无监督学习无标签无反馈。这决定了两者的目标导向和评估方式完全不同。
Ch03 回顾
四种学习方式中，监督和无监督是最核心的两种；半监督介于两者之间，部分有标签、部分无标签。
Terms
Supervised监督学习，有标签数据驱动的学习方式
Unsupervised无监督学习，无标签数据驱动的学习方式
Clustering聚类，按相似性将样本自动分组

机器学习的建模流程

机器学习建模是一个五步迭代流程。评估不佳时需要回退调整，不是一次性的线性过程。

1
获取数据 搜集与任务相关的数据集。类型包括经验数据、图像数据、文本数据等。关键：数据需匹配实际需求 —— 如中国房价模型不适用于美国。
2
数据基本处理 缺失值删除或填写、异常值离群点检测。为后续特征工程准备干净数据。
3
特征工程 五大子领域：提取、预处理、降维、选择、组合。让模型更容易学到规律。这是耗时耗精力最多的环节。
4
模型训练 根据任务选择合适算法：线性回归、逻辑回归、决策树、SVM 等。
5
模型评估 用测试集评估模型。效果好则上线服务；效果不好则回退迭代，重复上述步骤。
关键洞察
数据和特征决定 ML 上限，模型和算法只是逼近这个上限。数据处理和特征工程占大部分时间。
Ch04 / Ch05 回顾
Ch04 详细展开了五步流程；Ch05 展开了特征工程为何是最耗时环节及其五大子领域。
Terms
Pipeline建模流程，从数据获取到模型评估的完整链路
Iteration迭代，评估后回退调整再重新训练的循环过程

对特征工程的理解

特征工程：利用专业背景知识和技巧处理数据，让 ML 算法更容易学到规律、效果更好。好的特征工程直接决定模型天花板。

特征工程包含五大子领域，它们是并列关系：

特征提取 特征预处理 特征降维 特征选择 特征组合
子领域 做什么
特征提取 从原始数据提取任务相关特征，转成特征向量
特征预处理 解决量纲问题，让不同特征对模型影响一致。方法：归一化 (MinMaxScaler)、标准化 (StandardScaler)
特征降维 高维压缩到低维，保留主要信息。如 PCA 将多个特征融合为少量主成分
特征选择 根据专业知识挑选任务相关的特征子集，不改变原数据，去除无关特征减少噪声
特征组合 用乘法或加法合并多个特征为新特征。如 BMI = 体重 / 身高²，引入领域知识
鸢尾花案例
为何不用颜色作特征？专业人员判断颜色对分类的区分力不够。特征选择依赖领域知识 —— 不是特征越多越好。
Ch05 / Ch06 回顾
Ch05 展开了五大子领域的详细内容；Ch06 指出特征工程做不好可能导致欠拟合，做过度可能引入噪声导致过拟合。
Terms
Feature Engineering特征工程，处理数据使模型更易学习的技术总称
MinMaxScaler归一化，将数据缩放到固定范围如 0-1
StandardScaler标准化，将数据转换为均值 0 标准差 1
PCA主成分分析，一种特征降维方法

模型拟合问题及产生的原因

拟合：用在机器学习领域，用来表示模型对样本点的拟合情况。终极目标是泛化能力 —— 在新数据上也表现良好。

欠拟合 Underfitting
表现：训练集差，测试集也差
原因：模型过于简单、参数太少、表达能力不足
如 y=kx+b 拟合非线性分布
对策：增加模型复杂度、增加有效特征、换更强模型
过拟合 Overfitting
表现：训练集好，测试集差
原因：模型太复杂、数据不纯含噪声、训练数据太少
模型记住了噪声和个例，非真正规律
对策：正则化、异常值检测、特征降维、数据增强

Loss 曲线诊断：通过 train-loss 与 test-loss 的走势判断拟合状态：

奥卡姆剃刀
给定两个相同泛化误差的模型，倾向选择较简单的模型。简单模型往往泛化能力更强、更不易过拟合。
Ch01 / Ch06 回顾
Ch01 的 y=wx+b 是最简单的线性模型，参数少时可能欠拟合复杂数据；Ch06 详细讲解了 loss 曲线诊断和泛化概念。
Terms
Fitting拟合，模型对样本分布的模拟情况
Generalization泛化，模型在新数据上的表现能力
Occam's Razor奥卡姆剃刀，同等效果选更简单模型
Regularization正则化，约束模型参数防止过拟合
